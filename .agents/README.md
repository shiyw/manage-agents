# Manage Agents Runtime Workspace

This directory is the local workspace for Codex-managed note and filesystem operations.

- `index/`: rebuildable machine indexes for Obsidian, local knowledge-base, workspaces, documents, downloads, and selected external volumes.
- `logs/`: operation logs for interactive and automation runs.
- `trash/`: global quarantine area for files when no more specific managed-root trash applies.

Runtime data in these folders is intentionally ignored by git. Keep only the README files tracked.
